# Big-data-project
Project that will include the usage of Big Data technologies which will have following steps:
  * Data collection and preparation for storage.
This phase involves gathering data from various sources and preparing it for efficient storage and further processing.
  * Data storage in systems that enable large-scale processing.
Data is stored in platforms designed to support distributed processing and analytics.
  * Design and implementation of analytical scenarios.
This stage focuses on defining and executing analysis workflows, which may include the application of machine learning models
